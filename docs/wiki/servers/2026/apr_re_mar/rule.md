# 4月鯖 ルール

## 1 産業 (3つの街)
- 当初は同じ街でクエストを進め、一定地点で以下の3つの街に分裂して発展させます。各街には専売ライセンスが付与されます。
- 具体的には「初期の自動化」チャプターの「黒鋼レベルの機械力」を達成した時点で分裂させることとします。

### 農業街
- 野菜などの農作物の生産、肉や牛乳など畜産物の生産を行う。
- ライセンス：高度な調理、Create収穫機、有機素材の供給。

### 工業街
- 鉱石資源やその加工、製品の自動化を行う。
- ライセンス：GT加工機械の所持、ツール製造・修理、エネルギー供給。

### 商業街
- 加工された物品の販売などを行う。
- ライセンス：街間輸送サービス(Create鉄道)、自販機・中央市場管理、希少資源の仲介。

## 2 経済
- Lighman's Currencyのコインを公式通貨とします。以下は日本円換算の目安です。
    - 1c = 10円
    - 1i = 100円
    - 1g = 1000円
    - 1e = 10000円
    - 1d = 100000円
    - 1n = 1000000円
- 各社には初期の給付金として5nが配布されます。
- 各個人には給付金として4dが配布されます。
- 取引ルール：
    - 各街は専売品を販売し、不足する資材は他エリアから購入して経済を回してください。
    - アイテムの売買は原則として商業街が管理する中央市場の自販機(Vending Machine)を介して行います。
    - 自販機の設置および場所の貸し出しは商業街の許可が必要です。

## 3 建造物
工場や倉庫を含むすべての設備(3-1)は以下の要件を満たす場所に作成される必要があります。
階が1bでも地表以下にある場合その階は地下階となります。
- 地上階の場合はかならず地上からY+100以下に建築する必要がある。
- 地下階はかならず地上階と同じ、またはそれよりも小さいサイズでなければならない。
- 地下階はY24以下にあってはならない。（※採掘用の通路・シャフト等は除く）
- すべての建造物は地下階のみで構成することはできない。

## 3-1 設備
設備とは具体的に以下のものを指す。
- Createの機械設備を含む工場
- Createのネットワーク等によって使用される倉庫
- GregTechの機械およびマルチブロック設備

## その他
- Createのからくりを使用した、整地目的を除く露天掘り等の採掘行為全般を禁止とします。

以上のルールは今後変更される可能性が高く、常に目を通しておくことを推奨します。
